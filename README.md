# [Bloom '81] Coin Flipping by Telephone
## A Protocol for Solving impossible problems 

This is the Python implementation of the [Bloom coin-flipping protocol.](https://www.cs.cmu.edu/~mblum/research/pdf/coin/)

### Sample run

$ python3 bob.py
